## 1. Market Position & Competitive Analysis

AdvanceClip is a **cross-platform clipboard manager** with features that no single competitor combines:
- Cross-device sync (PC ↔ Android, LAN + Global)
- Drag-and-drop shelf UI
- Global hotkeys (Alt+1-0 instant paste)
- File/media transfer
- Keyboard navigation with Enter-to-paste
- Hover preview popup
- AI-powered context sorting

### Competitors

| Competitor | Price | What AdvanceClip does better |
|---|---|---|
| **Ditto** (free) | Free | No cross-device sync, no drag shelf |
| **ClipClip** | Free | No mobile companion, no LAN sync |
| **Paste** (Mac) | $3.99/mo | Mac-only, no PC |
| **1Clipboard** | Free (dead) | Abandoned, no active development |
| **CopyQ** | Free | Developer-focused, complex UI |

## 2. Monetization Strategy: Freemium Model

### Free Tier
- Local clipboard history (unlimited)
- Alt+C shelf with arrow key navigation
- Alt+1 through Alt+3 quick paste (limited to 3 slots)
- Basic text/image clipboard
- Single device only

### Pro — One-time $9.99 or $2.99/month
- **Alt+1 through Alt+0** (all 10 quick paste slots)
- **Cross-device sync** (PC ↔ Android, LAN + Global)
- **File/media drag-and-drop** transfer
- **Unlimited shelf history** (free = last 25 items)
- **Hover preview popup**
- **AI features** (smart categorization, context sorting, smart paste)
- **Priority support**

### Why One-Time Purchase Works Better
- Clipboard managers are **utilities**, not services
- Users hate subscriptions for small tools
- One-time purchase removes friction → higher conversion
- Subscription option available for those who prefer it
